Generic functions

Generic types are not only for classes. Functions can have them too: 


    fun <T> doSomething(item: T): List<T>

The function in the example has a generic type T. The generic type is always defined before the function name.

When calling a generic function, you specify the type arguments after the name of the function, e.g.: 


    val myCat = Cat()
    val myCatList = doSomething<Cat>( myCat )

You can read more at Kotlin reference
