List

A list is not created using a constructor, but through the method listOf(). E.g. 


    val myList = listOf(1,3,4)

However, the standard List in Kotlin is immutable. Which means methods to add and remove elements do not exist. Once the List is created, its objects cannot be added or removed.

To create a mutable list, you use the method mutableListOf(). The result is a List that does have the methods to add and remove elements

Standard operations on a List are as following (note: list indices, start at 0) 


    val myList = mutableListOf("A" , "B" , "C" )
    var item = myList[1]  // get the 2nd item == "B"
    myList.size //returns the number of elements


    //only when mutable
    myList[1] = "X"  //replace the 2nd item, only when mutable
    item = myList[1] // == "X"
    myList.remove("X") // remove "X" from List
    myList.removeAt(0) // remove whatever is at index 0
    myList.clear() //remove all elements
