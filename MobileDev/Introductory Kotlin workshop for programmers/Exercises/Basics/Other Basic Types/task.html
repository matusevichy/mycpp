Other Basic Types

Besides the String, there are of course plenty other basic types. We will quickly go over them.

Characters

A single character. Defined using single quotes (').

Example of the declaration of a character:


    val myChar = 'A'

Booleans

A boolean has two value, true or false. Supported operators are:

  • && : and
  • || : or
  • ! : not

    val myTruth = true || false

Numbers

Kotlin recognizes the following Number types:

  • Double
  • Float
  • Long
  • Int
  • Short
  • Byte

You will most likely recognize all of these.

Example of the declaration of a long:


    val myLong:Long= 123

Arrays

A container of certain objects that has a fixed length. Initialization example: Initialization is done using the arrayOf() function.

You can get anD set from/to the array using [i], where i is the index starting from 0.

Example: 


    val myArray = arrayOf(3,2,5,7)
    myArray[2] = myArray[1] + myArray[3] // assign 9 to second index
    // myArray is now [3,2,9,7]

You can read more at Kotlin reference

Assignment

Add all values of the array myArray together and store them in index 0 of myArray
