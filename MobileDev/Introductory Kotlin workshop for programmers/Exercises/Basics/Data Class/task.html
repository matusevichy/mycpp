Data classes

In Kotlin we can write a Data Transfer Object (DTO) with the usual functionality and utilities that will fit on one line!

This is called a data class and is marked as data:

data class Dto(val id: UUID, val data: String)

The compiler automatically derives the following members from all properties declared in the primary constructor:

  • equals()/hashCode() pair;
  • toString() of the form "Dto(val id: UUID, val data: String)";
  • and more..

Assignment: Write a Person data class

Now we like you to write a data class. Copy the following class. Rewrite it so it becomes a data class 


class Person{

    val name: String
    val age : Int
    val length : Double

    constructor( name : String,  age : Int, length : Double ){
        this.name = name
        this.age = age
        this.length = length
    }

    override fun equals(other: Any?): Boolean {
        if (this === other) return true
        if (javaClass != other?.javaClass) return false

        other as Person

        if (name != other.name) return false
        if (age != other.age) return false
        if (length != other.length) return false

        return true
    }

    override fun hashCode(): Int {
        var result = name.hashCode()
        result = 31 * result + age
        result = 31 * result + length.hashCode()
        return result
    }

}
