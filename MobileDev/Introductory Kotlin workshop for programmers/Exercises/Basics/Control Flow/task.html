Control Flow

There are several mechanisms for controlling the flow of your application:

  • if
  • when
  • for
  • while

If

Just your classic if statement. Kotlin does not support ternary operators (condition ? then : else). An example: 


    if( someTest() ) {
        println("Test successful")
    } else {
        println("Test failed")
    }

When

The when expression replaces the commonly used 'switch' statement. It looks as following: 


    when (myValue) {
        1 -> print("myValue is 1")
        2 -> print("myValue is 2")
        else -> { // Note the block
            print("myValue is neither")
        }
    }

You can use arbitrary expressions, not just constants, as branch conditions. For example: 


    when (myObject) {
        is Car -> print("myObject is a Car")
        is Plane -> print("myObject is a Plane")
        else -> { // Note the block
            print("myObject is an UFO")
        }
    }

When matches each condition in order and executes the first one that matches. If none match, else is executed.

For

A for loop can range over anything that provides an iterator (collections and such). Example of a for loop: 


    for (someVal in myCollection) {
       println(someVal)
    }

The code block is optional if it contains only one expression.

Ranges

To iterate over a range of numbers, you can use a range expression. E.g. 


    for (i in 1..10) {
       println(i)
    }

This prints 1 to 10 (including 10)

For more about ranges, read this Kotlin reference

While

Another classic. The while loop. Kotlin supports as well while as do ... while


    while( x > 0 ){
        x--
    }


    do {
        y--
    } while (y >0)

Break/Continue

And Kotlin supports the age-old break and continue operators in loops, to either abort the loop, or start the next iteration respectively.
