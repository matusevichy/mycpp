String templating and Raw String

Kotlin has support for Raw Strings and String Literals.

Raw Strings provide you the ability to use newline and spacing without escaping. A raw string is delimited by""" (triple quote) Eg.: 


    var myRawString = """first line
     second line
     third line"""

resulting in 


first line
     second line
     third line

String Templates is the ability to use variables directly in a String. Eg.: 


    var myVar = 123
    println("The value of myVar: $myVar")

resulting in 


The value of myVar: 123

For more complex expressions in the template, you can use ${ ... } (with accolades).
