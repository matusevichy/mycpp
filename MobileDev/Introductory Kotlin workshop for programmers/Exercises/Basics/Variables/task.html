Variables

Kotlin has two type of keywords to define variable declarations:
  • val - can only be assigned once
  • var - can be re-assigned

Syntax

Example of declaring a variable: 


    var myVariable:Int = 123

As you can see, the type of the variable is after the variable name, seperated by a colon (:).

Types can be inferred, so the following is allowed as well:


    var myVariable = 123

Kotlin will infer that myVariable is of type Int

Variables can be declared inside functions, as well as outside. The latter called 'top-level' variables.

For more about this topic, read this Kotlin reference.
