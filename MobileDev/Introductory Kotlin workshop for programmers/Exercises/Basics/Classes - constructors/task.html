Class definitions

Classes are defined using the keyword class. An example of a class declaration: 

    class Person(age:Int) {
    }

This is a class Person with a constructor that sets the field age . A class consists of 3 parts:

  • class name - the name of the class
  • class body - the code between the curly braces
  • class header- everything between the class name and body

if the body is empty, the curly braces can be omitted :


    class Person(age:Int)

To have secondary constructors, you define these using the keyword constructor. E.g. 


     class Person(age:Int) {
        constructor(age:Int, name:String) : this(age){
            println("I am the second constructor")
        }
    }

In this example the second constructor delegates to the first constructor, using the this keyword

To instantiate a class, just call it as if it is a regular function, e.g. 


    val john = Person(25, "john")
