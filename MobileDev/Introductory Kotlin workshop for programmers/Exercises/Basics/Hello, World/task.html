To get started we will describe a bit about functions.

Functions are declared using the fun keyword. Functions that are defined inside a class or object are called member functions. Values are returned using the return keyword. 


    fun myFunction():String{
        return "Cowabunga"
    }

This is a function returning a String, as defined by the :String after its name

If a function body contains only a single expression, you can reduce it to a single-line function.


    fun myFunction() = "Cowabunga"

Function parameters are provided between the "()". They always have the pattern name:Type 


    fun myFunction(myParameter:Int):String{
        return "Cowabunga nr" + myParameter
    }

To call a function, just call its name and pass the arguments between the (). 


    myFunction(12)
