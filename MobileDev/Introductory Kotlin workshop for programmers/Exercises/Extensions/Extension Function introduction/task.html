Extension Functions To declare an extension function, we need to prefix its name with a receiver type, i.e. the type being extended. The following adds a prefix function to String:

fun String.prefix() = "prefix> $this"

The this keyword inside an extension function corresponds to the receiver object (the one that is passed before the dot). Now, we can call such a function on any String:

"my great string".prefix()

and this will return 

prefix> my great string

Create a hello() extension function for String that will return "hello xyz", where "xyz" was the original string.
