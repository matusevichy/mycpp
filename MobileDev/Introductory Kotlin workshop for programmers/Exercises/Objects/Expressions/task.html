Object expressions


Object expressions can come in handy when we want to create an object of a slight modification of some class, without explicitly declaring a new subclass for it. This we call creating an object of anonymous class.

Here we have an example of an object expression:


open class A{
    open fun someFun(){
        println("hello")
    }

}

fun foo() {
    val adHoc = object : A(){
        override fun someFun(){
            println("is it me you're looking for?")
        }
    }
 }

Code in object expressions can access properties from the enclosing scope:




interface B{
    fun counter()
}

class FooBar() {

    var count = 0

    val adHoc = object : B{
        override fun counter(){
            count++
        }
    }
 }

You can read more about Object Expressions at: Kotlin reference.

Assignment

PrintShop has a method printDocument that accepts a document and a Printer. Printer is an interface.
A PrintShopOwner has a printedDocument property and a printIt function.
In order to print something, a PrintShopOwner will have to call the printDocument function from the PrintShop.

Objective:

  • create an object from the Printer interface and pass it into the printDocument method.
  • override the print method
  • assign document to printedDocument variable.
