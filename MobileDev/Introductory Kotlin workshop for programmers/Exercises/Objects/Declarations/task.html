Object Declarations


When we need to implement an interface without explicitly declaring a new implementation class or we want to declare a singleton object, in Kotlin we can do this by simply declaring an object.

An object declaration looks like this :

    
        object Utility{
            fun formatter(){
                ...
            }
        }

Or with a supertype : 

    
        object Utility : BaseUtilities{
            fun formatter(){
                ...
            }
        }

Couple of things to note :

  • it always has a name following the object keyword
  • an object declaration cannot be local (i.e. be nested directly inside a function).

You can read more about object declarations in the Kotlin reference.
