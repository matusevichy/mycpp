Functions

Functions in Kotlin are first-class citizens. This means it can be passed as an argument, returned from a function, modified and assigned to a variable.

Functions are declared using the fun keyword. Functions that are defined inside a class or object are called member functions. Values are returned using the return keyword. An example: 

 
        fun add(left: Int, right: Int): Int {
            return left + right
        }
This is a method accepting two parameters, 'left' and 'right', each of the type Int. It also indicates that it will return an Int.

Arguments

Named Arguments

When calling functions, the arguments can be named. e.g, instead of calling

add( 12, 15 )
you can call 
add( left=12, right=15 )
or 
add(  right=15, left=12 )

Default arguments

Default arguments can be defined using the = after the type along with the value. e.g. fun add(left:Int, right:Int = 0 ) will allow you to use the add function without a second argument. If done so, the second argument will have the value 0.

Variable number of arguments

A function can have a variable number of arguments, using the vararg keyword. 


    fun asList(vararg messages:String): List<String> {
    val result = ArrayList<String>()
    for (body in messages) //messages is an Array
        result.add(body)
    return result
}

Inside the function, the vararg parameter is seen as an array.

Note: if a vararg is not the last argument in the list, arguments following it, can be passed by name
