Infix functions

Example of the usage of an infix function looks like this: 


    myDog eats "munchies"

eats is in this example an infix function. It is a member function of myDog and receives one argument "munchies".

An infix function is a function that

  • Is either a member function or extension function
  • Has a single parameter
  • The parameter is not a vararg nor has a default value
(Extension functions will be discussed later on)

To define the infix function above, you would do the following: 


    class Dog{
        infix fun eats(food:String) = println("Eating $food")
    }

By preceding the function with the keyword infix and can be used as an infix function.

An example of an infix function that is already present within Kotlin would be the to function (used for example when creating a Map) :
"a" to 7
This will call a to() method on String to produce a Pair object containing "a" and 7. (This also involves extension functions and generics)
